# AppCadastroEscolar

Projeto desenvolvido em ASP.NET Blazor Server versão .Net8.0
O sistema consiste em um cadastro de Alunos Turma e Matricula;
Desenvolvido em camadas: Web (visão), Domain (classes, regas de negócio), Infraestrutura (acesso a dados);
A classe de Context, contém o seed inicial para as tabelas e o Identity;
É necessário executar a Migração para criar o banco de dados; 
Somente o CRUD base para a entidade Aluno está funcionando.
O busca CEP e a leitura CSV está pendente implementação.
